# haystack-rag-showcase

## Local setup with access to disk (more powerful)

### Get data

put api_key in .env

`cd tum_crawler`

`scrapy crawl tum`

### Process data

at root, run `docker-compose up`

run the whole scripts/with_scrapy.ipynb
